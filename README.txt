== Description ==

This Plugin converts a WordPress site to Single Page Application.


== Installation ==


1. Upload `wp-spa.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Will this work with any WordPress plugin? =

Yes, hopefully.

= What libraries does this use? =

Angular bundled with webpack.
